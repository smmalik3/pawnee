# Pawnee Frontend App

React + TypeScript + Vite frontend for the Pawnee Smart Civic Engagement Desk.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Required Environment Variables

- `VITE_AWS_REGION`
- `VITE_API_BASE_URL`
- `VITE_COGNITO_USER_POOL_ID`
- `VITE_COGNITO_CLIENT_ID`
